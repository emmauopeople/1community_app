# 1community_app
this application is a MVP which will further build to completion.
it is a skill up for cameroon where individuals can create skill listing say hair dressing
tailor, men's dressing, capenter, etc, with 3 images of what they can do.
Users can search say bed, capenter, chair, braid, wig, men suit, using location near me, all or enter a city
this return listing related to their search, and they can select a listing and contact the skill provider through whatsapp or email
